Toronto Festivals & Events

This is an open data feed of festivals and events in Toronto, transformed into Schema.org Event format for use in structured data applications.

Available Feeds

  • All Events (JSON-LD)
  • Upcoming Events (JSON-LD)
  • Daily Event Files (JSON Lines)

Usage

To use this data:

  • Download the full all.jsonld or just upcoming.jsonld
  • Each file is valid JSON-LD — can be parsed directly by JSON-LD processors
  • Daily snapshots (by date) are available in daily_jsonl/ in JSON Lines format
